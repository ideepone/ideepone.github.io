假设这个世界是可以理解的，那它的理解之处在哪儿？

发表于 2023-01-06 更新于 2024-08-13 分类于 时光集
本文字数： 756 阅读时长 ≈ 3 分钟

理解

The most incomprehensible thing about the world is that it is comprehensible.
这个世界的最不可理解之处就在于它是可以理解的。
— Albert Einstein

阿尔伯特·爱因斯坦他对这个世界的「理解」到底是一个什么认知？

是量子力学，随机概率，抑或其他什么？原本爱因斯坦可能没有说过这句话，只是媒体美化过了，但这个「理解」的「域」到底在哪儿？

他说，“上帝不掷骰子”。

但微观粒子的行为就是概率决定的，随机、概率就是这个世界的常态。

宏观世界其实也是一样，无数的行为决策背后，都是各种事件、条件错综复杂地影响结果。

苏格拉底

所有人都会死；

苏格拉底是人；

所以苏格拉底也会死。

在归纳和演绎逻辑的背后，如果突然加了一道概率。

99%的人都会死；

苏格拉底80%是人；

所以苏格拉底会死吗，死的可能性又有多大？

随机不等于不确定性

随机性和不确定性最大的差别就在于，这个事件可能出现的结果是否可知。

随机性是这个事件可能出现的结果我都知道，只是不知道下一次会出现哪个结果。

我曾以为这是个过程思维，就是当你经历过，你能清晰地看透事物发展全过程，透过历史长河，甚至用上帝视角去理解的时候，很多事都是顺其自然。

但很遗憾好像不是这样的，好比为什么存在多种意识形态、阵营的存在，甚至我们地球不过可能也只是一种随机的存在。

我们对未来的期望，试图理解未来的不确定性，最重要的一点是要不断地把不确定性转变为随机，因为这个世界大部分的事儿是随机的，而不是不确定的。

上帝可能喜欢掷骰子，但我们如果知道骰子只有1-6点那是幸运的，但如果我们不知道上帝掷骰子的点数会不会超过6点，这就是不确定，甚至听起来有些恐怖。

概率的终点是1

概率有三个性质：

第一，概率永远在0和1之间，不可能是负数。

第二，样本空间里所有基本事件概率之和是1。

第三，某个随机事件不发生的概率，等于减去这件事发生的概率。

第二条其实很有意思，概率之和是1，是100%，好像很显然，这个1的意思其实有两层。

首先，我们所有的样本要全，不要遗漏，但现实中很难做到；

其次，概率是系统之下的必然事件，只不过是细节之下的偶然事件。

如何让这个事件尽可能的发生，就是要深入了解了。

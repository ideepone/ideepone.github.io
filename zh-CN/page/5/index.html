教育的本质

发表于 2022-04-01 更新于 2024-08-12
本文字数： 961 阅读时长 ≈ 3 分钟

「教育」

前两天看到一个短视频讲「教育的本质是什么」，我觉得挺好，分享出来：

———

当年米开朗基罗做成大卫之后，自己也觉得震撼，所有人都震撼。

记者就问米开朗基罗，您是怎么创作出如此伟大的作品？

他没有轻易地回答，他说让我想一想，他想了半天之后很慎重的回答：

「不是我创造出这么伟大的作品，是那个石头里头本来就有个大卫，我只是把它释放出来。」

这话太精彩了，这话才是千古教育的本质。

「每个人身体里都有一个更好的自我，甚至是伟大的自我，每个孩子里都一定有这样一个自我。」

———

对于教育来讲，我觉得需要从两个角色看：教者和学者。

教者，老师也好，长辈也好，我喜欢的是鼓励和发掘的方式，就像视频里面说的，那个石头里面本来就有一个「大卫」，对于教者我们应该首先相信自己只不过是一个引路人，扮演的是一个传承的角色，未来一定是年轻人的天下，这是人类物种写在基因里面的。

很多教者喜欢用命令的方式，以身份来压制，这种很不好，年轻人是有自我意识的，是会从不同的角度思考问题的，我们应当以平等的方式去沟通，去针对事情本身的逻辑来沟通。

教者之所以是教者，很大可能是我们先人一步而已，有更多的过往经验；抑或是我们在某些方面的能力更强更专业，但这些「经验」或「能力」并不一定是对的，我们需要拿出来探讨。也正是因为这些，教者希望学者最终学到这些经验，以创造更多的价值。

而对于学者，其实最重要的一点是「自信」，要相信自己是独一无二的，是优秀的，当然自信不是盲目的，不能自负。这种自信也是建立在自我认知不断学习的基础上，建立在科学辩证的认知态度之上的。

我觉得两点认知很重要：

一是成功是建立在多次失败的改进基础之上，是需要刻意练习的。

越是有价值意义的事情，其实背后付出就可能越多，阻力就越大，这个世界我一直认为轻而易举获得的事情太少，偶尔有只是幸运值爆棚，但人生不能完全寄托于小概率事件，是需要不断的经历尝试，让自己在多个领域的全方位能力越来越强，越来越有生存发展的能力，这种能力是需要不断的与时俱进，不断地学习，活到老学到老，学到老活到老。

二是自己和环境给予的心理正向反馈，比如赞美、鼓励。

很多时候事情并不是百分百能成功，只是可能会成功，但积极心理学的潜意识某种程度是可以让自己坚持下来的，坚持坚持就成功了。不过有时候也要考虑「沉没成本」，就是如果事情付出太久太多仍然没有成功，是不是考虑要再过多的去投入了，创业时很容易犯这种错。

教者也好、学者也好，首先应该对自我的认知越来越清晰，其次对他人、对组织的认知也越来越清晰，要发现那个真正的自我，也要学会挖掘他人的「真我」，才可以让人类的文明得以传承下去。

历史·先辈

发表于 2022-03-28 更新于 2024-08-12
本文字数： 488 阅读时长 ≈ 2 分钟

那些先辈伟人，哲学家和科学家为什么会那么厉害？

  1. 文明是阶梯发展的，每一个阶段不同伟人解决的问题都是站在前辈巨人的肩膀之上。这也是为什么人类要不断的发展，需要传承的核心原因之一。

比如：正是因为苏格拉底，亚里士多德这些前辈的哲学理性思维，才奠定了整个逻辑基础，后续的科学都是以逻辑为基础。

  1. 我觉得人类，不管哪一代人都在思考几个问题：

a). 我是谁？关于自我认知的一个探索。

b). 我和你一起是谁？社会组织关系的一个探索。

c). 宇宙是什么？ 大到天体物理，小到微观粒子，对世界的一个探索。All in one。

个人觉得思维方式是可以被锻炼的，是有顶层的，符合第一性原理，认知是需要积累的，随着不同时代，远古时代、石器时代、机械时代、电器时代、信息时代，每一个时代被发现或者被解决的问题依据当时的科技生产力，不断的被推进。

个体放在人类发展的洪流里面，伟人是必然出现的，是环境的产物。只是为什么他们是伟人，我们还不是，我到现在也认为是时代带来的科技红利和顶层思维方式的认知觉醒。最近刚把爱因斯坦《我的世界观》也翻出来读，也在尝试着思考，在他们那个时代，他们在做什么，他们为什么会这样思考？

我非常喜欢爱因斯坦说的那句话，这个世界是可以被理解的。

我们还有很多问题待解决，还有很多科学家在不断的研究探索认知这个世界的本质问题。

维度｜点·线·面｜运动

发表于 2022-03-26 更新于 2024-08-12
本文字数： 500 阅读时长 ≈ 2 分钟

蚂蚁的眼里，世界也许是平的， 人类以前也以为地球只是平的。

线是由一个又一个点连接而成的，面是一条又一条线组成的。

我们常说的「升维」、「降维」，到底这个维度是什么？

山顶为什么会一览众山小，人类为什么很容易嘲笑虫子的可悲，甚至当你回首看看从前的自己，甚至觉得无知可笑。

我们很容易在意一次考试的好坏，一次创业的成败，一次人生的悲喜，这些重要吗？很重要，因为这是你人生阅历的那条条线，那个立体的你。

「物质是运动的。」

这是一条非常基本的定律，就像我们经常说，要学会动态的看问题，很简单的发现，当你把点从A移到B，这个就是一条线；当你把一条线从A移到B，这个就是一个面；当这个面从A移动到B，这就是一个立体。

「物质运动是需要一个力的，而在阻力情况下运动是需要消耗能量的」。

有一天咸鱼突然意识到，我要有梦想了，这个「意识」，无论是外在的还是内在的驱动，都是一种「力」，于是朝着这个梦想去了。

梦想是什么？梦想是一次人生的跃迁，是一次低维度向高维度奔赴的决心和历程。

阻力是什么？是你要到达高维度的无数次能量消耗，是人生种种的经历。

人生本该是立体的，所以需要好奇心，需要不断的探索，需要不断的经历。

人生就是一场修行，一场追求彼岸的过程。

当明白很多时候，学会动态地观察问题的时候，全面立体多维度的，甚至更高维度的观察问题的时候，其实或许很多事情就知道了答案。

共生

「共生」

  1. 如果这个世界只剩人类，人类会不会最终走向灭亡？

  2. 当全世界只剩你一个人的时候，门外依然会传来敲门声，这个时候你会选择开门还是不开门？

其中第二个问题是爱因斯坦提出来的，但这里我想表达的是，如果当你醒来，全世界都不理解你的时候，或者你无法理解这个世界的时候，你会什么感受？

上周我终于入坑了《宝可梦·阿尔宙斯》，刚好里面也提到了宝可梦的世界观，人类和宝可梦是共生的关系。

我们人类是宝可梦的朋友，不需要那种上下级之类的关系。

从我的理解，人类之间也好，还是人类与地球的其他物种之间也好，还是与某种未知的文明，其实都是共生的一种关系。

新冠爆发以后，很多人其实是期盼着疫情早早结束，我个人一直认为，新冠是不可能彻底结束的，人类要考虑的应该是如何与病毒共存。

这个世界人类其实并不是主宰，反而某种程度，微生物或者病毒才是这个世界的主宰，人类不过是一个宿主而已。

在团队协作过程中，我非常喜欢张一鸣提到的两个原则：

  1. 公司内部禁止称呼「总」。

  2. 公司内部不可以用「您」。

很多时候，我们寻求的是共同发展，每一个人在团队组织当中都不过是一个角色，这个角色无关孰重孰轻，关心的应该是大家怎么一同通过每个人的能动性把一个目标漂亮的完成。

明白了这点，就能理解老板也好，创始人也好，员工也好，都不过是一个角色，而每一个角色都应有自己的本质工作。

很多年以来我就不喜欢在大公司上班了，更不喜欢体制内，不是他们不好，是我觉得不适应，我喜欢独立个体的最大化表达的氛围。

最近在思考另外一个问题，「个人成长和组织成长」。

我给自己提出了一个假设：就是信奉组织的成长是伴随着个人成长的推进，进而一起螺旋发展。

那又如何让个人和组织共同成长了？

有的时候是「集体智慧」，有的时候是「乌合之众」，这中间的本质差异是如何的呢？

无论如何，我觉得共生是唯一的出路，这个世界单兵作战生还的个体太少了，寒冬季节大家都期望抱团取暖，大环境也一样。

这其实让我又想到了另外一个问题，是否有一个自我大统一的完全循环的终极系统呢，因为「熵」值永远是需要被新的能量平衡的，这外来的能量到底是哪儿来的呢？

也许在边界趋近无穷大之前，我们唯有的就是不断地假设外界能量源源不断。

所谓共生，目前看来是不断的通过获取外在的能量，来寻求内在的平衡与发展。

事实·观点·情绪

发表于 2022-03-13 更新于 2024-08-12
本文字数： 570 阅读时长 ≈ 2 分钟

「事实·观点·情绪」

我们在观察理解世界、描述沟通的时候，可能到底在理解什么，表达什么？

事实和观点都需要表达的，情绪某种程度在表达的过程中，你的用词以及肢体语言都可能会自然体现出来。

事实分「事物」和「事件」，事物是一个独立的个体，事件是由一系列事物的作用产生的效应。

观点是你对某个事实的一种描述和看法。

举个例子：

有一天早上，你在上班路上看到了一只狐狸，并且手机拍下了照片，然后你立即跟朋友很兴奋地分享了这个过程。

你发消息说，我今天路上竟然看到了一只狐狸。

朋友可能会回答你，真的吗？在城市里面怎么可能有狐狸？

立马你又发了拍的图片，来确定你告诉的信息是真的。

于是，你和朋友就这件事热烈的展开后续的讨论。

我们来看下这一次的描述过程中：什么是事实，什么是观点，什么是情绪？

狐狸还有狐狸的照片是事实。

基于你看到了一只狐狸的描述分享，你朋友问你，在城市里面怎么可能有狐狸这是观点。

兴奋、疑惑、惊奇(竟然)是情绪。

其实，我们在描述表达过程中，主要就是这些，但事实可靠吗，被描述过的事实的可靠吗？人有时候很容易陷入主观表达的情绪当中。

比如，如果你看到了狐狸后，但没有拍照，你朋友问你的时候，你没有那张照片，你会怎么办，你朋友会相信吗？

一般是朋友的话基本是无条件信任你，甚至会给你建议，下次一定要拍个照片。但如果这是分享给一个陌生人，这个结果就不一定了。

这个过程其实蕴含着「逻辑」和「批判性思维」，例子也相对简单，但当我们面对更复杂事情的时候，我们就更应该清晰的了解哪些是事实，哪些是观点，又附带了哪些情绪。

隐性假设·暗物质

发表于 2022-03-13 更新于 2024-08-12
本文字数： 776 阅读时长 ≈ 3 分钟

「隐性假设·暗物质」

有一阵子看逻辑学后对「隐性假设」很感兴趣，后来想想就是「现象」和「本质」的关联关系，那如何透过现象看到本质，是否一定能看到本质？

我们先说「隐性假设」，很多时候我们在表达的时候附带了太多的隐性假设，这种隐性假设可能来源于个人的知识积累，认知层次，甚至于某一个环境。

日常生活中所说的「常识」、「文化」其实都是一种很容易被忽略的隐性假设合集。

举个例子：

我从小到大吃的豆腐脑都是甜的，后来在北方读大学，有一天吃早餐的时候，发现有豆腐脑，于是就点了一碗，等上来的时候就傻眼了，竟然是咸的还带卤带香菜。

在北方，豆腐脑就是咸的，而我的常识里面，一直认为豆腐脑是甜的，等上来要吃的时候，就蒙了。

当然，现在很多地方服务员会问你，咸的还是甜的，或者我自己主动说要甜的。

这其实就是常识或者文化的一个差异导致的。

这些隐性假设很不容易显现出来，导致我们在表达沟通、推断做决策的时候出现偏差。

我们又如何能把一件事表达或者理解的全面、准确、透彻、可靠呢？

其实「理解」更重要，因为很多事只有你理解了才能继续沟通，很多时候别人没有义务把事情说得那么透彻明白，甚至是有意刻意的隐藏关键信息。

这里面其实常用的方式是「批判性思维」，或者简单的一点是「追问」，把自己的疑惑表达出来，直到清晰，甚至再用自己的话术表达一遍，才可能真正的理解了。

比如，我们在做产品的时候，会特别注重名词概念的解释，要保证大家的理解是一致的；或者在沟通的时候，别人给你抛出一个概念或问题的时候，不清晰的问题需要确认一下，甚至复述一下问对方是不是这样的意思，等等。

那为什么我要把「隐性假设」和「暗物质」一起呢，主要是拿来做一个比喻。

暗物质是理论上提出的可能存在于宇宙中的一种不可见的物质，它可能是宇宙物质的主要组成部分，但又不属于构成可见天体的任何一种已知的物质。

据说，这个世界暗物质占比80%以上，其实生活中很多事背后的推动有可能是那些暗物质决定的，只是你不知道而已。

很多时候，我们应该尽可能的去挖掘、批判、分析、理解那些事情背后的隐性假设，而这些隐性假设才可能是真正的本质，那些显性的只是表象。

「偏见」

带着偏见看世界：每个人都是带着成见看待世界的，如果不带着成见，那你对世界根本就没有看待的方式。

我很喜欢十三邀这个观点，因为我觉得这个是对「世界观」非常好的一个诠释，虽然直到现在很多人都在探索这个世界原本的真实性。

但其实这个不重要，重要的是我们作为一个人降落到人间后，你个人的视野就是这个世界有一个独特的视角，是唯一的、是差异的、是值得表达的。

这个世界原本的色彩可能不仅仅是光线的五彩斑斓，可能也是因为你赋予了某些事物的感情色彩，甚至是偏见，进而精彩的。

当然，我们期待的这些偏见是正向的，是积极的。

2.「小报童」是对「Flomo」的一个延续。

我真的很喜欢Flomo是在于它对碎片化思考沉淀方式做到了极致克制，轻量而快捷。

我极力推荐自己身边朋友在使用Flomo的同时，我也在思考Flomo的分享到底会怎么做，但「小报童」没出来之前，我也跟小伙伴探讨，Flomo在我看来，这个产品闭环没有做完。

直到前段时间我看到「小报童」的出现，我觉得Flomo这个闭环终于通了，并且惊奇少楠是以一种新的产品形态表现出来，真的很佩服。

在我看来「小报童」解决了我的两个痛点：

  1. Flomo的个别观点我真的会忍不住分享给我的一些好朋友，但我如果一张图一张图分享就太麻烦了。

  2. 如果这些观点是经过一些沉淀和思考的，而这些观点恰好有一些志同道合的朋友也感兴趣，而我的视角节省了他们对现有信息噪音的一个筛选服务，并且愿意一起分享讨论，那么我觉得这些服务就是有价值的，也是值得持续付出的。

3.「雕琢自我 普惠他人」

这八个字是「笔记侠」的价值观，也是对我既往到未来价值观的一种强烈认同。

人生是一个持续学习的成长过程，需要不断地自我雕琢，但随着年龄的增长，会发现这种成长之下的积累和沉淀，如果恰巧能用笔记或者文章的形式，分享给周遭的朋友或者年轻的小伙伴，又恰巧刚好是一个视角、刚好是一个线索、刚好有一些帮助，也许这辈子做不到「普惠」，但刚好对身边的人有一些帮助，就是值得的，也是幸运的。

「线索」

「十三邀·刘擎」期里面讲了很多，让我记忆深刻的一个概念是「线索」。

「你去看我访谈、演讲，我都会特别强调这只是一个线索，是通向更开阔的、更深厚的、更重要的那些著作的桥梁，我希望架起这个桥梁，我是给你一个导游图。」

生活中的很多事实，别人给你的信息也好，自己做得一些事情结果也好，不过都是一个对未来的线索。

当你预想自己是「活在未来」，有一个终极谜底在等待你揭开的时候，或许你会对这个世界充满了好奇和热情。

生命放在整个历史长河里面，本是千丝万缕的联系构建的，确定论也好，概率论也好，关键是不断的朝着你心中的那个终极谜底，抽丝剥茧，不断地探索。

物种的多样性一定要够，不能单极化

发表于 2022-06-05 更新于 2024-08-12
本文字数： 350 阅读时长 ≈ 1 分钟

「多样性」

今天把《圆桌派·尹烨》又看了一遍，信息量很大，择其中一个视角分享一下：

“物种的多样性一定要够，不能单极化。比如番茄，如果全世界只栽培一种番茄，只有栽培种，没有野生种，那么只要这个病毒感染一次，这个就没了。”

《星际穿越》里面有一个场景，偌大的玉米地也是因为一次病毒就全没了。

人类基因已经测序完毕，总共30亿个碱基对，但现在具体哪个碱基对控制什么还不完全清楚。

生物的多样性能很好的保证物种的延续性，基因的自私性也可能意味着抵抗力的脆弱性，这也能很好的解释为什么「共生」才应该是生态的最终选择。

比如，很多对抗很多疾病病毒，我们都是从一些植物、动物种提炼出来然后研制成药或者疫苗才得以人类对抗，这是幸运的。

如果有一天只有人类了，动植物的生态多样性单一化，也就意味着人类的灭亡。

其实对于人类文明及个体也是一样，如果一味地只是个体文明的传达，没有开放的包容和学习态度，自然会被时代淘汰掉。

本质是必然的，现象是偶然的

发表于 2022-06-04 更新于 2024-08-12
本文字数： 423 阅读时长 ≈ 2 分钟

「必然·偶然」

很多人说上海终于解封了，这个「终于」透露着无限的期盼和无奈，最终有了结果。

可事实不过是，上海解封了，这是必然的，6月1号不过是个时间问题，这两个月不过是一个过程。

好比，宇宙始终会朝着无序发展，这是必然的。

对于大部分人来说，会经历出生、学生、成人、结婚、生子、死亡等等一些过程，这是必然的，偶尔有些人是非必然的，无法出生，无法读书、无法结婚生子等等。

这对大众是偶然，对个例才是必然。

我有一个很基础的观点，这个事儿只要你用心去做了，基本都会有结果，无论结果是好是坏，不过是达到多少预期。

过程是单向运动不可逆的，运动是有起点和终点的，这是必然的，只不过什么时候是起点，什么时候是终点，这是非必然的。

现象是偶然的，本质才是必然的。

大部分人都热衷沉浸于现象当中，甚至沉溺于某一次的成功与失败。

其实，过去是必然的，未来才是偶然的。

又好比，大部分人一定是会有朋友的，但朋友是不是你，这个是偶然的。友情也好、爱情也好，都难能可贵，是因为茫茫人海中，得一知己或者相遇相知相爱都是偶然的，是概率的，是值得庆幸的。

理性是必然的，感性才是偶然的。

偶然是文艺浪漫的，必然是无趣无味的。

无所用，无所不用

发表于 2022-05-15 更新于 2024-08-12
本文字数： 862 阅读时长 ≈ 3 分钟

「天道」

周末把《天道》电视剧又下来开始看了，今天看到关于「天道」的一段话：

———

丁元英说：“透视社会依次有三个层面：技术、制度和文化。小到一个人，大到一个国家一个民族，任何一种命运归根到底都是那种文化属性的产物。强势文化造就强者，弱势文化造就弱者，这是规律，也可以理解为天道，不以人的意志为转移。”

“强势文化就是遵循事物规律的文化，弱势文化就是依赖强者的道德期望破格获取的文化，也是期望救主的文化。强势文化在武学上被称为“秘笈”，而弱势文化由于易学、易懂、易用，成了流行品种。”

芮小丹说：“从字面上能理解一点，但知道又如何，怎么用呢？”

丁元英说：“无所用，无所不用。”

———

这个「无所用，无所不用」恰好是我这几天思想瓶颈的一个非常好的答案吧。

先说「天道」，这个可能就是我一直以来想寻找的规律，不以意志为转移的东西，可以是定律、思维、或者其他种种。

长久以来的探索，我确定并相信了《形式逻辑》这种底层驱动，以及对《马哲》这种世界观的认同。

确定逻辑和马哲作为底层驱动力之后，我迷惑了好些天，也许是苦苦探究的终于有了一个答案，也或许是终于达到阶段性的彼岸。

可即使掌握了形式，具体的内容还是需要突破的，我一度在想，要不要继续去研究具体的内容，但天下种种，哪怕有十足的好奇心和精力，也有终将耗尽的时候，想到这一层，那么一瞬间，顿感无趣和无力了，甚至失去了目标，也可能跟上海长时间遥遥无期的封闭有影响。

我迷惑的是，因为这些有种形而上，看似解决不了衣食住行的问题，不禁会问逻辑和哲学能带给我们什么呢？

芮小丹也提了非常好的这么一个问题：“有什么用呢？”

无所用，这个看起来没什么用，而后面一句「无所不用」恰巧是我一直以来相信的，只是没坚定下来的。

无所不用的边界就在于，逻辑和马哲其实是可以应用在日常生活工作的方方面面。

我跟朋友说，我30多岁了好像现在终于悟出了点门道，是幸运吧。

好在，再回过来，天道给了我一个明确的答案吧，其实答案自己早已知道，只是需要一个去相信和坚持的理由吧。

有个很有意思的问题：人类做很多事都需要找到某种意义，但这种意义真的重要吗？

直到现在我也不知道，但很多事对我来说是需要搞清楚为什么的。

所谓「无所用，无所不用」，其实是大道至简，看似简单的东西要做到其实很难，而那些复杂的东西，其实是可以很简单，只因为未知或把它想复杂了。

老子说：“无为而无不为”。

顺应「天道」，实事求是，见路不走，即见因果。

「阴·阳」

古语云：“一阴一阳之谓道”。

有时候追求极致、念念不忘的思考好处就是证明自我的一个存在吧。

使用Flomo构建自己知识体系的时候经历了几个阶段：

  1. 经典摘抄，少感悟，随机关键词标签无分类。

  2. 多感悟，少摘抄，有分类标签[P.A.R.A]，结构化表达。

  3. 思考事实·观点、理性·感性、客观·主观、显性·隐性、自我·他人、个人·社会、有序·无序、正·反等等。

  4. 思考形式·内容，思考逻辑、追踪哲学、思考概念。

  5. 思考此岸·彼岸，思考大道至简、追求探索山水境界。

  6. 思考抽象·演绎。

  7. …

直到现在思考也谈不上到了那个心中追求的那个道的顶点，抑或能触碰到那个门的边界，每每阅读看到先辈们早在几百上千年就总结的精辟透彻，喜悦之极甚至感动的哭泣，倍感幸运吧。

《铿锵行天下第二季》有一个假设： 假如地球上没有了沙漠会产生怎样的后果？

好像沙漠恢复成森林是一件不错的壮举，但沙漠的存在带来是全球气候的一个循坏，如果沙漠变成了森林，那可能对地球就是一个新的生态灾难。

正义和罪恶是绝对的吗，对和错真的重要吗，黑与白中间其实是有灰的，有理数竟然不是终结，无理数有尽头吗？能量为什么会守恒的，熵值为什么会不断的增大等等。

计算机的世界不过是0和1，现实可能不过是一阴一阳，相互轮转，相互更迭。

也许真实和虚拟不过一念之间，什么又是那个量到质的边界点呢？

念念不忘，必有回响

发表于 2022-05-07 更新于 2024-08-12
本文字数： 716 阅读时长 ≈ 3 分钟

「重生」

哲学有一个悖论问题，忒修斯之船，如果忒修斯的船上的木头被逐渐替换，直到所有的木头都不是原来的木头，那这艘船还是原来的那艘船吗？

有的说是，有的说不是，如果每天我们的细胞系陈代谢，有一天体内的细胞都替换了，我还是我吗？那如果我们每天思考认识都在更新，我们还是我们自己吗？

问题还在那儿，是不是原来的船不重要，新的生活总要继续。

如果说物质决定意识，意识作用于物质来说，一个新的认知或许会带来我们的一次「重生」。

英文「重」的词根是「re」，比如reborn、rework、rebuild、review、restart、recall等等，都是表示再、重新、重复的意思。

互联网里面经常说的一个机遇是，当新的一次科技生产力的出现，任何行业都可以被重新做一遍。

互联网从web1.0到web2.0，还有最近一直在炒的web3.0，大到各个行业都值得「RE」一遍。

那对于个人呢，我们很多认知是后天被灌输的，是从小到大的环境影响的，但这一些都不一定是对的，人类的习惯是行为决定的，行为是意识决定的。

那我们是不是值得把我们过往的知识、概念、认知都需要重新审视一遍呢？

也许这是一个漫长的过程，但一些重要的观念我们是值得审视的，最近几年一直在探索到底哪儿出了问题，直到最近似乎从两本书找到了源头有了答案，而且有意思的是这两本竟然都是教材，一本《形式逻辑》，另外一本是《马克思主义哲学》，也或许正是因为大学对这两门课有过好奇和接触，留了念想，才有今天追根溯源的一个路径吧。

简单的讲是三观的一次重新审视：世界观、价值观、人生观。

马哲里面讲到一个道理，人与世界的关系主要包括两方面：一是认识世界，二是改造世界。人类所有的活动不过都是围绕这两件事在做的，这也就是为什么现在我反过来看「雕作自我、普惠他人」是一个什么样的底层和价值驱动。先是认识自我、认识世界，然后是改造世界。

每个人都会有新生的机会，而这机会，需要的是对自己、对这个世界的一次重新审视，对三观的一次洗礼。

在我的价值观里面，只有对自己和世界的深刻认知，才能换来一个安心的彼岸之路。

习惯是个行为，行为更容易改变

「标签·定性」

前两天抖音上看到《此念官方》的一个心理学方面的短视频《什么样的话最不应该对孩子说》，很不错。

嘉宾黄仕明这样回答的：在身份层面负面的语言（不会说）。

举个例子，如果我有一个个案跟我说，嘿，老师，我是一个酒鬼，他找我做心理咨询。

我不会认同的，我会说，嘿，你是说，你有喝酒的习惯吗？

习惯是个行为，行为更容易改变。

如果这个人说，我是一个抑郁的人。

我会这样说，你是说，你笑得比别人少一点吗？

如果我的孩子跟我说，爸爸，我肯定会失败的。身份认同，我不可能成功。

我会说，噢，你是说，这一次，你会失败吗？

让那个绷紧的地方（深吐气），打开一个放松的空间，在这个空间里才可能有新的东西进来。

当然，明天考试可能还是会失败的，但是我们之间有一个同盟，一起有一个意图，看到未来如何可以做得比现在好一点点，我成为了孩子眼里那个同盟，相信我的那个人。

———

很多时候，不管是给自己或者给他人很容易打上性质的标签，比如，我是一个loser，他是一个坏人等等。

可能因为这一个性质的标签，接下来若干年都在背负着这个无形的压力，甚至让人透不过气来，如果幸运，能走出来很好，但如果走不出来，可能就一直是个死循环。

人生是没有边界的，是没有那么多标签的，更不应该定性，这个世界也不是非黑即白，非好即坏的，很多时候时间是最好的陪伴，积极的心态会改变很多的。

我思，故我在

发表于 2022-04-23 更新于 2024-08-12
本文字数： 877 阅读时长 ≈ 3 分钟

「思考」

笛卡尔说过：“我思，故我在。”

很多年前我在知乎上问过一个问题，黑格尔说的「存在即合理」怎么理解，这么多年好像就是追寻一个问题，既然我的存在是合理的，那为什么我会存在，又是因为什么我才存在与不同。

人生有时候不过是很多个假设，这些假设可能是社会的，也可能是自我的，剩下的就是围绕这些假设无尽的探索与追寻。

这些天在读两本书《直击本质》和《思考的力量》，起源是我对「本质」这个概念的源头感兴趣了，我们大多时候都是希望看到事物的本质，但却很难看到本质，那为什么有的人能看到本质，有的人看不到呢？

大多人都知道思考很重要，什么独立思考、批判思考、深度思考、逆向思考，然后又会提炼很多种思维方式，可人类文明至今一直没能把思考这件事解释清楚，比如意识是什么？

《思考的力量》里面对概念有一句这样的解释，思考指的是对外界的一个或者多个对象进行分析和判断，并得出关于该对象的具体结论的思维过程。

我一直在思考的一个问题是「如何理解这个世界」？

一个从世界到个人的反映过程，一个从客观世界到主观世界的过程，一个从此岸到彼岸的过程，很难但值得一生去思考。

这个世界你看到的可能不是真的，需要不断的确定什么才是真的，有的人觉得无趣，但我觉得这个很有意义。

生物学上讲，思考本质上是神经细胞之间发生的神经冲动，这些神经冲动带来了反射和其他行为。

我们每天都在思考，大量的信息涌入让你被动地思考，遇到新的问题主动地思考，甚至好像有时候心不在焉无意识地思考。

那到底怎么思考，怎么让自己的思考更高效，怎么让自己的思考变成可执行可操作呢？

思考是没有尽头的，但是可以有维度的，有深度、广度的，思考也可以是有边界的，有阶段的，甚至思考可以自我反省，思考思考的思考，元思考。

有时候理解世界的规则很简单，从事物本身到大脑的具体反应，可就是这短短的一段路，要经历的太多太多，一个不留意可能就没了边际，无法达到你心中的彼岸。

人类文明得以延续在我看来是前人不断思考探索的结晶，有几时我也在想，当你懂得了前人伟人的思考方式，他们的意识形态在这些千千万万的概念关系中被完全拷贝了一份，那个具有跟他们一样思维的人，还是你吗？

叔本华说过，“思考的人，在精神王国中，等于一国的君主”，愿每一个懂得思考的人能主宰自己的世界，成为自己的王。

配角还是主角？

「角色」

人生其实是一场又一场的角色扮演之旅。

很多人第一次蜕变的经历是突然知道了，原本以为自己是故事的主角，不过是个配角，甚至只是个路人而已。

小时候看电影喜欢看主演，慢慢得看电影除了看主角，更喜欢看配角，了解小角色的成长经历。

如果人生是一场戏，导演就只给你10秒钟，你会怎么把握呢？

我们一生其实会扮演很多个角色，可很多时候我们并不了解角色，甚至冥冥中很多角色是被突然安排过来的。

出生下来成为儿子女儿，上学成为学生，工作成为职员，然后成为高管老板，然后恋爱了结婚了，成为情人爱人，然后成为了爷爷奶奶外公外婆，抑或成为很多很多不同的角色。

可真的能明白每一个角色本身背后的意义和责任呢？

我工作中对角色感悟最深的经历是第一次当总监的时候，升职加薪内心是高兴的，可当真正开展工作中，对上对下的管理，前面几个月真的是糟糕透顶，很多事还是喜欢亲力亲为，也很难把握轻重缓急，老板说你，手下还顶撞你，团队被带得晕头转向的。

后来有一天突然明白了，到底「总监」是做什么的，这个角色是一个什么样的角色，然后清晰的把边界分清楚，专注做本质工作慢慢就好起来了。如果很难一下把角色换过来，自己和同事都会很累的。

什么是员工，什么是管理者？

真正的管理者，是你愿意主动的去思考、去担当、去引导、去行动，而不是命令，你真正做得好的能力有多少可能比得过一线的专业者呢。

那生活中呢，什么是子女和父母，什么是学生和老师，什么是朋友，什么是爱人，这些角色的边界到底在哪儿呢，又期望一个与对方交往的时候是一个什么角色呢？

有太多的时候，我们是角色混淆了，或者是角色不清晰，是被动的入选了某个角色，其实可以主动的担当起来。

前阵子网上有个短视频，在国外的一个战争游戏中，红蓝双方抢夺阵地，其中一方节节败退，最后指挥官直接下线跑了，这个时候一个中国玩家主动出来中英双语指挥，带领着大家最后走向胜利。

也许这个中国玩家开始玩只是简单的想扮演一个士兵，但关键时刻却能主动承担指挥官角色并且完美地胜任下来。

这里面最主要的一点是日常「用心」，用心去观察，用心去思考，敢于尝试和担当，关键时刻才能胜任。

人生很多角色就是被安排的，那就好好的把这个角色扮演好，如果有一天导演给了一个10秒的角色，你会把握住吗？

王晶评价周星驰在大话西游里面说10秒换了7个表情，这7个表情背后又付出了多少呢？

「痛感」

痛感指痛苦的感觉或深切地感动。

成长是一个不断走出舒适区的过程，长久地待在舒适区，会让人失去思考的习惯，会让人逃避现实，是一种缺少危机意识的行为。

薛定谔说过：“人活着就是在对抗熵增定律，生命以负熵为生”。

这是我最喜欢的一句话，悲观而浪漫。

保持痛感是对生命生活的一种敬畏，生命本身就是「向死而生」，生活本就是「无常」，只有真正明白了这些才会从内心迸发出一种向生的力量感。

看电影、听音乐、阅读、锻炼等等很多种方式，让内心和身体都可以处在一个「心跳」的状态。

保持痛感，保持感悟，向阳而生，未来才会越来越精彩。

上海疫情封闭的这些天

发表于 2022-04-08 更新于 2024-08-12
本文字数： 668 阅读时长 ≈ 2 分钟

「无常」

世人都在求安定，无奈世事总是无常。

上海疫情看似来得很突然，连封城好像都猝不及防，每天都有新增感染者，原定5号解封的，今天8号还在待定，很多人囤粮都没了，甚至压根儿就没囤，社会的快递外卖等派送力又不够，小区居委会的组织管理一下又没跟上来，若大的上海好像乱糟糟的。

不确定的大环境之下，其实最重要的是抗风险能力，反脆弱的能力。

「每一个事物都会从波动中得到利益或蒙受损失，脆弱性是指因为波动和不确定而承受损失，反脆弱性则是指让自己避免这些损失，甚至从混乱中不确定中获利。」

首先，我觉得反脆弱的能力重要的一项是「心态」，积极乐观的心态是非常重要的，就像《反脆弱》书中也提到尼采的那句话：“杀不死我的，只会让我更强大。”

这场抗疫中，有不少人在抱怨在吐槽，说这也不行那也乱，而有担当的人早已置身一线分析问题解决问题。

大多困难面前失败的根本原因是自己的意志消沉和心态的崩塌。

其次，我觉得要有「危机意识」。

比如囤粮，有的家庭就会囤不少，这个囤不是疯狂抢购，是有节制的囤。如果5号说解封，如果不解封，你会怎么做，我至少多囤点，以防万一。囤一些必要持久的，如米油盐蛋等。

我一直相信一个原则，任何事情要做最坏的打算。

工作中，你的团队告诉你项目进展10号能完成，你自己心理要有预期，如果10号完不成怎么办，你是否要预留时间，或者提前赶进度。

新冠疫情的这几年，多少行业公司都无法坚持下来直至倒闭关门，而新东方却反而能活下来，俞敏洪从第一天开始就做好了这个准备。

德州扑克其实很锻炼这个，都是概率，偶尔赌一赌可以，但如果总是入池，或者动辄All in，结果只有输。

无常其实是常态，影响事情的发展因素是多维的，是概率的，百分百太难了，要学会的是在无常和不确定性下很好的生存发展，这是一项基本能力。

可以读读《反脆弱》，也深入了解一下这项能力。
